## Introduction

### What is Slide-CN?
Slide-CN is a simple library to create presentations and decks using react components. You can customize, and make these components your own to create beautiful presentations. Open Source, Open Code. Slide-CN is built using pure react components, so it will work with any framework of choice.

### What does Slides as Code mean?
Slides as code simply means to create a Slide deck using code and not drag and drop editors like Canva or PPT. There are numerous benefits to this:
- Components are consistent across slides
- Your slides look good out of the box
- You can deploy it as a website and share your slide deck with people easily
- You can use advanced version control systems like git on your presentations
- Your presentations can be dynamic. Since it is simply code, you can get realtime content for your presentation
