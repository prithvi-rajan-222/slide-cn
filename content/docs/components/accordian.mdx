## Overview
This is the [Accordian](https://ui.shadcn.com/docs/components/radix/accordion) from Shadcn.

## Installation
<InstallCommand packageName="accordian" />

## Usage
This component is useful to add a lot of information to your slide when space is a constraint. It can also allow you to dictate where the viewers focus is during a presentation.
