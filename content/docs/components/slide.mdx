## Overview
This is the main Slide component. Use this as the root component for each of your slides. You can add a background to your slide by passing a component to the `background` prop. You can also add a footer to your slide by passing a component to the `footer` prop. See [SlideFooter](./slide-footer) for more information.

This is an easy way to add a background to your slide. Any react component can be used as a background.
For this example I used the `PixelSnow` component from [ReactBits](https://reactbits.dev/components/pixel-snow).
