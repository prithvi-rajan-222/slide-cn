The slide-cn skill gives your agent the component rules it needs to build decks with the right layout components, file structure, and slide patterns.

## Installation

Run this in your project:

```bash
npx skills add prithvi-rajan-222/slide-cn
```


## Usage

Once installed, ask your agent for the deck you want:

> "Create a 5-slide pitch deck for a dev tool startup"

> "Add a slide with a code demo on the right and explanation on the left"

> "Replace the benefits slide with reveal cards"

The skill covers the core slide-cn components, layout choices, deck structure, backgrounds, and file organization.
