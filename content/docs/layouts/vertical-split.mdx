## Overview
Use this component within a slide to split any content vertically.

## Installation
<InstallCommand packageName="slide-cn@vertical-split" />

## Demo
<ComponentDemo className="h-[400px]" description="Split content vertically with a customizable ratio">
  <VerticalSplit ratio={0.7} className="h-full rounded-lg overflow-hidden">
    <VerticalSplit.Top className="bg-gradient-to-br from-orange-50 to-amber-100 dark:from-orange-950 dark:to-amber-950 flex items-center justify-center p-8">
      <div className="text-center">
        <h3 className="text-2xl font-bold mb-2">Top Section (70%)</h3>
        <p className="text-muted-foreground">This section takes up 70% of the vertical space</p>
      </div>
    </VerticalSplit.Top>
    <VerticalSplit.Bottom className="bg-gradient-to-br from-cyan-50 to-blue-100 dark:from-cyan-950 dark:to-blue-950 flex items-center justify-center p-8">
      <div className="text-center">
        <h3 className="text-2xl font-bold mb-2">Bottom Section (30%)</h3>
        <p className="text-muted-foreground">This section takes up the remaining 30%</p>
      </div>
    </VerticalSplit.Bottom>
  </VerticalSplit>
</ComponentDemo>

## Usage
```tsx
<VerticalSplit ratio={0.7}>
	<VerticalSplit.Top>
		This content will take up 70% of the available space on the top
	</VerticalSplit.Top>
	<VerticalSplit.Bottom>
		This will take up only the bottom 30% of the parent container
	</VerticalSplit.Bottom>
</VerticalSplit>
```
