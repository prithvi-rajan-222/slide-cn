## Overview

`SpotlightGradient` shines a bright radial "spotlight" from an off-centre position against a darker field. Great for drawing the audience's eye to a focal point.

| Variable | Description |
|----------|-------------|
| `--background` | Mid-range fill colour |
| `--background-2` | Outer edge colour |
| `--background-3` | Spotlight highlight colour |

```tsx
<Slide background={<SpotlightGradient x="30%" y="30%" />}>

```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `x` | `string` | `"30%"` | Horizontal position of the spotlight centre |
| `y` | `string` | `"30%"` | Vertical position of the spotlight centre |
| `size` | `number` | `60` | Spotlight radius as a percentage of the container |
| `className` | `string` | — | Merged with `absolute inset-0` |
| `style` | `CSSProperties` | — | Merged with the gradient. Pass CSS variable overrides here. |
