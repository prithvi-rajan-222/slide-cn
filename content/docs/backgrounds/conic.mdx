## Overview

`ConicGradient` renders a smooth colour sweep (conic gradient) that rotates through the slide-cn theme tokens around a configurable centre point.

| Variable | Description |
|----------|-------------|
| `--background` | Start / end colour of the sweep |
| `--background-2` | Quarter-turn colour |
| `--background-3` | Half-turn colour |

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `angle` | `number` | `0` | Starting angle in degrees |
| `position` | `string` | `"center"` | CSS position of the gradient origin (e.g. `"30% 40%"`) |
| `className` | `string` | — | Merged with `absolute inset-0` |
| `style` | `CSSProperties` | — | Merged with the gradient. Pass CSS variable overrides here. |
