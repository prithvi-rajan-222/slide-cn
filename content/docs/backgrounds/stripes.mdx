## Overview

`StripedGradient` tiles soft diagonal stripes across the slide using `--background` and `--background-2`.

| Variable | Description |
|----------|-------------|
| `--background` | Primary stripe colour |
| `--background-2` | Secondary stripe colour |

```tsx
import { StripedGradient } from "@/components/ui/slide-cn/backgrounds/stripes";

<Slide background={<StripedGradient angle={45} stripeSize={40} />}>
  {/* slide content */}
</Slide>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `angle` | `number` | `45` | Stripe angle in degrees |
| `stripeSize` | `number` | `40` | Total width of one stripe pair in pixels |
| `className` | `string` | — | Merged with `absolute inset-0` |
| `style` | `CSSProperties` | — | Merged with the pattern. Pass CSS variable overrides here. |
