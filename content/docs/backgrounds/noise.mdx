## Overview

`NoiseGradient` combines a linear gradient with an SVG-based film-grain texture overlay, giving slides a tactile, printed feel.

| Variable | Description |
|----------|-------------|
| `--background` | Gradient mid colour |
| `--background-2` | Gradient start colour |
| `--background-3` | Gradient end colour |

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `noiseOpacity` | `number` | `0.08` | Opacity of the noise texture layer (0–1) |
| `className` | `string` | — | Merged with `absolute inset-0` |
| `style` | `CSSProperties` | — | Merged with the container. Pass CSS variable overrides here. |
